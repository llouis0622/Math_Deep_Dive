# 1. 무리수

### 무리수 명제

- 제곱해서 2가 되는 유리수는 없음
    
    $(\frac{p}{q})^2 \neq 2$
    

### 수 체계

- 자연수 : $\mathbb{N} = \{1, 2, 3, 4, \cdots\}$
- 정수 : $\mathbb{Z} = \{\cdots, -2, -1, 0, 1, 2, \cdots\}$
- 유리수 : $\mathbb{Q} = \{\frac{p}{q}$ 꼴로 표현되는 모든 분수$\}$
    
    순서 정의 : $r < s, r = s, r > s$
    
- 실수 : $\mathbb{R}$
- $\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$
