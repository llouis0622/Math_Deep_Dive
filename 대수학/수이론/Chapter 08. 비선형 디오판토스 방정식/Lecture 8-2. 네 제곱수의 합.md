# 1. 세 제곱수의 합

### 세 제곱수의 합

- 정수 $m, n$에 대해 $4^n(8m + 7)$과 같은 형태의 양의 정수는 세 제곱수의 합으로 나타낼 수 없음

# 2. 네 제곱수의 합 항등식

### 네 제곱수의 합 항등식

- $(a^2 + b^2 + c^2 + d^2) \times (x^2 + y^2 + z^2 + w^2) = (ax + by + cz + dw)^2 + (ay - bx + cw - dz)^2 + (az - bw - cx + dy)^2 + (aw + bz - cy - dx)^2$

# 3. $x^2 + y^2 \equiv -1 \pmod p$ 풀기

### 비둘기집의 원리

- $k + 1$개 이상의 물건을 $k$개의 상자에 넣는 경우, 최소한 한 상자에는 그 물건이 2개 이상 들어 있음

### 네 제곱수의 합 보조정리

- $p$를 홀수인 소수라고 하면, $0 \leq x, y \leq \frac{p - 1}{2}$일 때 $x^2 + y^2  \equiv -1 \pmod p$를 만족하는 정수 $x, y$가 존재
- $p$가 홀수인 소수일 때, $mp$를 네 제곱수의 합으로 나타낼 수 있는 $m < p$인 양의 정수 $m$이 존재

# 3. 네 제곱수의 합에 대한 정리

### 네 제곱수의 합

- 모든 소수는 네 제곱수의 합으로 나타낼 수 있음

### 라그랑주의 네 제곱수의 합 정리

- 모든 양의 정수는 네 제곱수의 합으로 나타낼 수 있음
