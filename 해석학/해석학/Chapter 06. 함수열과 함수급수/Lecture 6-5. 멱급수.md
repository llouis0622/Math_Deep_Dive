# 1. 멱급수

### 멱급수로 표현된 함수

- $f(x) = \sum_{n=0}^\infty a_nx^n = a_0 + a_1x + a_2x^2 + a_3x^3 + \cdots$

### 멱급수의 절대수렴

- 멱급수 $\sum_{n=0}^\infty a_nx^n$이 어떤 점 $x_0 \in \mathbb{R}$에서 수렴하면 $|x| < |x_0|$를 만족하는 모든 $x$에서 절대수렴

# 2. 멱급수의 고른 수렴성

### 멱급수의 수렴성

- 멱급수 $\sum_{n=0}^\infty a_nx^n$이 어떤 점 $x_0$에서 절대수렴하면 이 멱급수는 닫힌 구간 $[-c, c]$에서 고르게 수렴

# 3. 아벨 정리

### 아벨의 보조 정리

- $b_1 \geq b_2 \geq b_3 \geq \cdots \geq 0$을 만족하는 수열 $b_n$과 부분합이 유계인 급수 $\sum_{n=1}^\infty a_n$에 대해 어떤 $A > 0$가 있어 모든 $n \in \mathbb{N}$에 대해 $|a_1 + a_2 + \cdots + a_n| \leq A$가 성립하면 모든 $n \in \mathbb{N}$에 대해 다음이 성립
    
    $|a_1b_1 + a_2b_2 + a_3b_3 + a_nb_n| \leq Ab_1$
    

### 아벨의 정리

- $x = R > 0$에서 수렴하는 멱급수 $g(x) = \sum_{n=0}^\infty a_nx^n$은 구간 $[0, R]$에서 고르게 수렴
- 멱급수가 $x = -R$에서 수렴할 때도 비슷한 결과 성립

# 3. 멱급수에서 가능한 연산

### 아벨 정리 명제

- 멱급수가 집합 $A \subseteq \mathbb{R}$에서 점별수렴하면 임의의 콤팩트 집합 $K \subseteq A$에서 고르게 수렴

### 멱급수의 고른 수렴 명제

- 모든 $x \in (-R, R)$에 대해 $\sum_{n=0}^\infty a_nx^n$이 수렴하면 미분한 급수 $\sum_{n=1}^\infty na_nx^{n-1}$ 또한 각 $x \in (-R, R)$에서 수렴
- $(-R, R)$에 포함되는 콤팩트 집합에서 고르게 수렴

### 멱급수의 무한 미분

- 멱급수 $f$가 구간 $A \subseteq \mathbb{R}$에서 수렴한다고 가정
    
    $f(x) = \sum_{n=0}^\infty a_nx^n$
    
- 함수 $f$는 $A$에서 연속이고 임의의 열린 구간 $(-R, R) \subseteq A$에서 미분가능
    
    $f'(x) = \sum_{n=1}^\infty na_nx^{n-1}$
    
- $f$는 $(-R, R)$에서 무한번 미분가능하며 각각의 도함수는 해당하는 급수를 항별로 미분하여 얻을 수 있음
