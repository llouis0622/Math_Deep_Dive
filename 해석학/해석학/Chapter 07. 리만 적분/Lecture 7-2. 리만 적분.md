# 1. 분할, 상합, 하합

### 분할

- 구간 $[a, b]$에서 $P$는 다음 부등식을 만족하는 $[a, b]$의 점으로 이루어진 유한집합
    
    $a = x_0 < x_1 < x_2 < \cdots < x_n = b$
    

### 상합과 하합

분할 $P = \{x_0, x_1, x_2, \cdots, x_n\}$의 각 부분구간 $[x_{k-1}, x_k]$에 대해 다음과 같이 두자

$m_k = \inf\{f(x) \mid x \in [x_{k-1}, x_k]\}, M_k = \sup\{f(x) \mid x \in [x_{k-1}, x_k]\}$

- 하합 : $L(f, P) = \sum_{k=1}^n m_k(x_k - x_{k-1})$
- 상합 : $U(f, P) = \sum_{k=1}^n M_k(x_k - x_{k-1})$

### 세분

- $Q$가 $P$의 점을 모두 포함할 때, $P \subseteq Q$이면 분할 $Q$를 분할 $P$의 세분이라 함

### 세분 명제

- $P \subseteq Q$이면 $L(f, P) \leq L(f, Q)$이고 $U(f, P) \geq U(f, Q)$

### 세분 보조정리

- $P_1$과 $P_2$가 $[a, b]$의 임의의 두 분할이면 $L(f, P_1) \leq U(f, P_2)$

# 2. 적분가능성

### 적분가능

- 분할을 정교하게 하여 상합과 하합이 같아질 때의 함수
- 상합과 하합에 극한을 취하기보다 완비성 공리를 이용해 상합의 하한과 하합의 상한을 같게함

### 상적분과 하적분

구간 $[a, b]$의 모든 가능한 분할의 집합을 $\mathcal{P}$라고 할 때

- 상적분 : $U(f) = \inf\{U(f, P) \mid P \in \mathcal{P}\}$
- 하적분 : $L(f) = \sup\{L(f, P) \mid P \in \mathcal{P}\}$

### 상적분과 하적분 보조정리

- $[a, b]$에서 유계인 임의의 함수 $f$에 대해 항상 $U(f) \geq L(f)$가 성립

### 리만 적분가능성

- 구간 $[a, b]$에서 정의된 유계인 함수 $f$에 대해 $U(f) = L(f)$이면 리만 적분가능
    
    $\int_a^b f = U(f) = L(f)$
    

# 3. 적분 판정법

### 적분 판정법

- 유계 함수 $f$가 $[a, b]$에서 적분가능할 필요충분조건은 임의의 $\epsilon > 0$에 대해 다음 부등식을 만족하는 $[a, b]$의 분할 $P_\epsilon$이 존재하는 것
    
    $U(f, P_\epsilon) - L(f, P_\epsilon) < \epsilon$
    

### 적분 판정법 명제

- $f$가 $[a, b]$에서 연속이면 적분가능
