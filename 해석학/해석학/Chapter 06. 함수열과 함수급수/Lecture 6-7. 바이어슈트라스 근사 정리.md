# 1. 바이어슈트라스 근사 정리

### 바이어슈트라스 근사 정리

- $f : [a, b] \rarr \mathbb{R}$는 연속함수라고 할 때, 상수 $\epsilon > 0$과 모든 $x \in [a, b]$에 대해 다음을 만족하는 다항함수 $p(x)$가 존재
    
    $|f(x) - p(x)| < \epsilon$
    

# 2. 보간법

### 다각형 함수

- 함수 $\phi$에 대해 각 부분구간 $[x_{i-1}, x_i]$에서 일차함수가 되는 $[a, b]$의 분할이 다음과 같이 존재한다고 하자
    
    $a = x_0 < x_1 < x_2 < \cdots < x_n = b$
    
- 다각형 함수 : 연속함수 $\phi : [a, b] \rarr \mathbb{R}$

### 보간법

- 주어진 점을 지나는 그래프를 가지는 함수를 찾는 과정

### 보간법 명제

- $f : [a, b] \rarr \mathbb{R}$가 연속함수라고 할 때, 상수 $\epsilon > 0$과 모든 $x \in [a, b]$에 대해 다음을 만족하는 다각형 함수 $\phi$가 존재
    
    $|f(x) - \phi(x)| < \epsilon$
