# 1. 소수의 중요성

### 소수와 합성수

- 소수 : 정수 $p$의 약수가 1과 $p$뿐일 때, 1보다 큰 $p$
- 합성수 : 1보다 큰 정수 중 소수가 아닌 수

# 2. 산술의 기본 정리

### 산술의 기본 정리

- 1보다 큰 모든 정수는 소수이거나 소수들의 곱으로 나타낼 수 있음

# 3. 소수의 성질

### 소수 명제

- $p$가 소수이고 $p|(a \times b)$이면 $p|a, p|b$

### 소수 따름정리

- 만약 $p$가 소수이고 $p|(a_1 \times a_2 \times a_3 \times \cdots \times a_n)$이면 $p|a_1, p|a_2, \cdots, p|a_n$
- $p, q_1, q_2, q_3, \cdots, q_n$이 모두 소수이고 $p|(q_1 \times q_2 \times \cdots \times q_n)$이면 소수 $q_1, q_2, \cdots, q_n$ 중에 $p = q_k$인 $q_k$ 존재

# 4. 산술의 기본 정리에 대한 증명

### 산술의 기본 정리 증명법

- $n$이 소수들의 곱이라는 것에 대한 증명(존재성)
- 유일성 증명

### 산술의 기본 정리 따름정리

- 1보다 큰 정수 $n$은 소수이거나 소수들의 곱으로 나타낼 수 있음

    서로 다른 소수 $p_1, p_2, p_3, \cdots, p_r$에 대해, $n = p_1^{k_1} \times p_2^{k_2} \times p_3^{k_3} \times \cdots \times p_r^{k_r}$


### 소인수분해

- $n = p_1^{k_1} \times p_2^{k_2} \times p_3^{k_3} \times \cdots \times p_m^{k_m} = \prod^m_{j=1}p_j^{k_j}$
