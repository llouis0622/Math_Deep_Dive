# 1. 최대공약수의 성질

### 유클리드 알고리즘(스테인 알고리즘)

- $ax + by = \gcd(a, b)$

### 베주 항등식

- 0이 아닌 $a, b$에 대해 $ax + by = \gcd(a, b)$를 만족하는 정수 $x, y$가 존재

### 유클리드 알고리즘 명제

$\gcd(a, b) = g$이면, 다음이 성립

- $d|a, d|b \Rightarrow d|g$
- $\gcd(a, b) = g$는 임의의 두 정수 $m$과 $n$에 대해 $ma + nb$의 가장 작은 양의 정수

$\gcd(a, b) = g$라 할 때, 임의의 양의 정수 $m$에 대해 다음이 성립

- $\gcd(ma, mb) = mg$

# 2. 서로소

### 서로소

- 둘 중 하나는 0이 아닌 정수 $a, b$가 $\gcd(a, b) = 1$을 만족

### 유클리드의 보조정리

- $\gcd(a, b) = 1, a | (bc) \Rightarrow a|c$

# 3. 유클리드 알고리즘 적용

### 유클리드 알고리즘

- $a$를 $b$로 나누는 나눗셈 알고리즘 적용
    
    $a = bq_1 + r_1$
    
- 만약 나머지가 $r_1 = 0$이라면, $a = bq_1, \gcd(a, b) = \gcd(bq_1, b) = b$
    
    $\gcd(a, b) = b$이므로 종료
    
- 만약 $r_1 \neq 0$이라면, $a = bq_1 + r_1$이고 $b$를 $r_1$로 나눔. 몫을 $q_2$, 나머지를 $r_2$로 하는 식 생성
    
    $b = r_1q_2 + r_2$
    
- 만약 나머지가 $r_2 = 0$이라면, 종료. 그렇지 않으면, $r_1$과 $r_2$에 나눗셈 알고리즘 다시 적용. 나머지가 0이 될 때까지 이 과정 반복
- 0이 아닌 마지막 나머지는 주어진 정수 $a, b$의 최대공약수

### 유클리드 알고리즘 명제

- $a = bq + r \Rightarrow g = \gcd(a, b) = \gcd(b, r)$
