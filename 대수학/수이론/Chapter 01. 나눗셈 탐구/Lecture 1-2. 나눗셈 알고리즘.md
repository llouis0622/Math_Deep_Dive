# 1. 나눗셈 알고리즘 절차 소개

### 몫과 나머지

- 몫 : $a = bq + r$에서 $q$에 해당
- 나머지 : $a = bq + r$에서 $r$에 해당

# 2. 나눗셈 알고리즘의 증명

### 자연수의 정렬성(WOP; Well-Ordering Principle)

- 공집합이 아닌 양의 정수의 부분집합에는 가장 작은 원소가 존재
- 일반성을 잃지 않음(WLOG; Without Loss of Generality)

### 나눗셈 알고리즘

- 임의의 정수 $a, b$에 대해, 다음을 만족하는 유일한 정수 $q$와 $r$이 존재
    
    $a = bq + r$
    
- 존재성 : 집합 $S = \{a - mb|m \text{은 정수이고} \ a - mb \geq 0\}$가 공집합이 아님
- 유일성 : 정수 $q, r$의 유일함
