# 1. 부분수열

### 부분수열

- 실수열 $(a_n)$과 증가하는 자연수 수열 $n_1 < n_2 < n_3 < n_4 < \cdots$에 대해 다음 수열이 부분수열
    
    $(a_{n_k}) = (a_{n_1}, a_{n_2}, a_{n_3}, a_{n_4}, \cdots)$
    

### 부분수열 명제

- 수렴하는 수열의 부분수열은 원래 수열과 같은 극한값으로 수렴

### 발산판정법

- 다른 극한값으로 수렴하는 두 개의 부분수열이 존재할 때 수열은 발산

# 2. 볼차노-바이어슈트라스 정리

### 볼차노-바이어슈트라스 정리

- 모든 유계수열은 수렴하는 부분수열을 가짐
