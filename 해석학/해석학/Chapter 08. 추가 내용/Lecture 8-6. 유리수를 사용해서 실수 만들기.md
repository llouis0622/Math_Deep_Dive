# 1. 실수

### 실수의 존재성

- 공집합이 아니고 위로 유계인 집합은 반드시 상한을 가지는 순서체가 존재하며, $\mathbb{Q}$를 부분체로 포함

### 완비성 공리

- 공집합이 아니고 위로 유계인 부분집합은 항상 상한을 가짐

# 2. 데데킨트 절단

### 절단

다음 세 가지 성질을 가지는 유리수의 부분집합 $A$

- $A \neq \emptyset$이며 $A \neq \mathbb{Q}$
- $r \in A$이면 $A$는 $q < r$인 모든 유리수 $q$도 포함
- 집합 $A$는 최댓값을 가지지 않음 → $r \in A$이면 $r < s$인 $s \in A$가 존재

### 실수

- 유리수 집합 $\mathbb{Q}$의 모든 절단을 모은 집합

# 3. 체와 순서에 대한 성질

### 체

- 두 연산 덧셈과 곱셈이 존재하는 집합 $F$이며 두 연산은 다음 조건을 만족함
- 교환법칙 : 모든 $x, y \in F$에 대해 $x + y = y + x$이고 $xy = yx$
- 결합법칙 : 모든 $x, y, z \in F$에 대해 $(x + y) + z = x + (y + z)$이고 $(xy)z = x(yz)$
- 항등원의 존재 : 모든 $x \in F$에 대해 $x + 0 = x$이고 $x1 = x$이며 $0 \neq 1$인 특별한 두 원소 0과 1 존재
- 역원의 존재 : 원소 $x \in F$에 대해 $x + (-x) = 0$인 원소 $-x \in F$가 존재
    
    원소 $x \neq 0$에 대해 $xx^{-1} = 1$인 원소 $x^{-1}$이 존재
    
- 분배법칙 : 모든 $x, y, z \in F$에 대해 $x(y + z) = xy + xz$

### 순서와 순서체

집합 $F$에서 순서는 $\leq$로 나타내며, 다음 3가지 성질을 만족

- 임의의 $x, y \in F$에 대해 $x \leq y$ 또는 $y \leq x$ 중 적어도 하나는 참
- $x \leq y$이고 $y \leq x$이면 $x = y$
- $x \leq y$이고 $y \leq z$이면 $x \leq z$

다음을 만족하는 순서 $\leq$가 주어진 체 $F$를 순서체라고 함

- $y \leq z$이면 $x + y \leq x + z$
- $x \geq 0$이고 $y \geq 0$이면 $xy \geq 0$
