# 1. 실수 집합 $\mathbb{R}$의 순서체 정의

### 실수 집합

- $\mathbb{R}$ : $\mathbb{Q}$를 포함하는 집합, 모든 원소가 덧셈에 대한 역원과 0을 제외한 원소는 곱셈에 대한 역원 가짐

### 완비성 공리

- 공집합이 아니고 위로 유계인 부분집합은 항상 상한을 가짐

# 2. 상한과 하한

### 유계

- 모든 $a \in A$에 대해
- 위로 유계 : $a \leq b$를 만족하는 $b \in \mathbb{R}$가 존재할 때의 집합 $A \subseteq \mathbb{R}$, $b$는 $A$에 대한 상계
- 아래로 유계 : $l \leq a$를 만족하는 하계 $l \in \mathbb{R}$이 존재할 때의 집합 $A$

### 상한과 하한

- 실수 $s$가 다음 두 조건을 만족할 때, $s$를 집합 $A \subseteq \mathbb{R}$의 최소상계라 함
- $s$는 $A$의 상계
- $A$의 임의의 상계 $b$에 대해 $s \leq b$
- 최소상계(상한) : $\text{lub} \ A$
- 최대하계(하한) : $\text{inf} \ A$

### 최댓값과 최솟값

- 최댓값 : 실수 $a_0$가 $A$의 원소이고 모든 $a \in A$에 대해 $a_0 \geq a$일 때 $a_0$
- 최솟값 : 실수 $a_1$이 $A$의 원소이고 모든 $a \in A$에 대해 $a_1 \leq a$일 때 $a_1$
