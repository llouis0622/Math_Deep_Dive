# 1. 집합

### 집합

- 원소라 불리는 대상의 모임
- $x \in A$ : 주어진 집합 $A$에 대해 $x$가 $A$의 원소
- 합집합 : $x \in A \ \text{or} \ x \in B \rarr x \in A \cup B$
- 교집합 : $x \in A \ \text{and} \ x \in B \rarr x \in A \cap B$
- 공집합 : $\emptyset$, 원소가 없는 집합
- 포함 : 집합 $A$의 모든 원소가 집합 $B$의 원소인 것, $A \subseteq B$
- 부분집합 : 포함 관계의 집합
- 여집합 : $A^C = \{x \in \mathbb{R} | x \notin A\}$

### 드 모르간 법칙

- $(A \cap B)^C = A^C \cup B^C, (A \cup B)^C = A^C \cap B^C$

# 2. 함수

### 함수

- 두 집합 $A$와 $B$에 대해 각 $x \in A$에 $B$의 한 원소를 대응시키는 규칙
    
    $f : A \rarr B$
    
- 정의역 : 집합 $A$
- 치역 : $f(x) = \{y \in B |$어떤 $x \in A$에 대해 $y = f(x)\}$

# 3. 논리와 증명

### 증명법

- 귀류법 : 간접 증명
- 직접 증명법 : 명백히 참인 명제에서 시작하여 엄밀한 논리적 추론을 거쳐 결론에 도달
- 대우증명법 : 논리적 모순에 도달하여 처음 가정을 부정해야 할 때까지 증명 진행

### 귀류법 예시

- 두 실수 $a$와 $b$가 같기 위한 필요충분조건은 임의의 양수 $\epsilon > 0$에 대해 $|a - b| < \epsilon$인 것
    
    <pf> (⇒) 만약 $a = b$이면 모든 실수 $\epsilon > 0$에 대해 $|a - b| < \epsilon$
    
    (⇐) 만약 모든 실수 $\epsilon > 0$에 대해 $|a - b| < \epsilon$이면 $a = b$
    

# 4. 수학적 귀납법

### 수학적 귀납법

- 만약 $S$가 $\mathbb{N}$의 부분집합이면서 다음 두 가지 조건을 모두 만족하면 반드시 $S = \mathbb{N}$
- $S$는 1을 포함하고
- $S$가 자연수 $n$을 포함할 때마다 $S$는 $n + 1$도 포함
