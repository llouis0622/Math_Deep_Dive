# 1. 완비성 공리 정리

### 축소구간성질

- 닫힌 구간열 $I_n = [a_n, b_n] = \{x \in \mathbb{R} | a_n \leq x \leq b_n\}$에서 각각의 $I_n$이 $I_{n + 1}$을 포함할 때, 다음과 같이 닫힌 구간열은 공집합이 아닌 교집합을 가짐
    
    $I_1 \supseteq I_2 \supseteq I_3 \supseteq I_4 \supseteq \cdots = \bigcap^\infty_{n=1}I_n \neq \emptyset$
    

# 2. 유리수 집합 $\mathbb{Q}$에서의 조밀성

### 아르키메데스 성질

- 임의의 실수 $x \in \mathbb{R}$이 주어지면 $n > x$를 만족하는 $n \in \mathbb{N}$ 존재
- 임의의 실수 $y > 0$이 주어지면 $\frac{1}{n} < y$를 만족하는 $n \in \mathbb{N}$ 존재

### 유리수 집합 $\mathbb{Q}$에서의 조밀성

- 두 실수 $a, b$가 $a < b$이면 $a < r < b$인 유리수 $r$ 존재

# 3. 제곱근의 존재성

### 제곱근 명제

- $\alpha^2 = 2$인 실수 $\alpha \in \mathbb{R}$ 존재
    
    <pf> $T = \{t \in \mathbb{R} | t^2 < 2\}, \alpha = \text{sup} \ T$
