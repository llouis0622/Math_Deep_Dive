# 1. 피타고라스 세 쌍

### 피타고라스 세 쌍

- $n \geq 3$일 때, $x^n + y^n = z^n$을 만족하는 양의 정수 $x, y, z$는 존재하지 않음
- $n = 1, 2$에 대해서는 해가 존재 → 피타고라스 정리를 만족하는 정수를 찾는 것

# 2. 두 제곱수의 합

### 두 제곱수의 합

- 모든 정수를 두 제곱수의 합으로 나타낼 수 있는가 → 불가능
- 모든 양의 정수를 세 제곱수의 합으로 나타낼 수 있는가 → 불가능

# 3. 두 제곱 항등식

### 두 제곱 항등식

- 임의의 정수 $a, b, c, d$에 대해 다음이 성립
    
    $(a^2 + b^2)(c^2 + d^2) = (ac - bc)^2 + (ad + bc)^2$
    

### 두 제곱 항등식 보조정리

- $p \equiv 1 \pmod 4$를 만족하는 소수 $p$와 $k < p$인 양의 정수 $k$에 대해 $x^2 + y^2 = kp$를 만족하는 양의 정수 $x, y$가 존재

### 소수의 두 제곱수의 합

- $p \equiv 1 \pmod 4$를 만족하는 소수 $p$는 두 제곱수의 합으로 나타낼 수 있음
- 홀수인 소수 $p$를 유일한 두 제곱수의 합으로 나타낼 수 있음 $\Leftrightarrow p \equiv 1 \pmod 4$

# 4. 두 제곱수의 합 정리

### 두 제곱수의 합 정리

- $m$을 양의 정수라 하고 서로 다른 소수 $p_i$에 대해 $m = p_1 \times p_2 \times \cdots \times p_r \times N^2$이라 할 때, 모든 소수 $p_j$가 $p_j = 2$이거나 $j = 1, \cdots, r$에 대해 $p_j \equiv 1 \pmod 4$인 경우에 $m$을 두 제곱수의 합으로 쓸 수 있음

# 5. 두 제곱수의 합으로 표현되지 않는 정수

### 두 제곱수의 합으로 표현되지 않는 정수

- $n \equiv 3 \pmod 4$이면 $n$은 두 제곱수의 합으로 나타낼 수 없음

### 두 제곱수의 합 정리의 역

- 서로 다른 소수 $p_j$에 대해 $m = p_1 \times p_2 \times \cdots \times p_r \times N^2$이라 할 때, $m$을 두 제곱수의 합으로 나타낼 수 있으면 소수 $p_j$ 중 어떤 것도 $j = 1, \cdots, r$에 대해 $p_j \equiv 3 \pmod 4$를 만족하지 않음

### 두 제곱수의 합 정리의 역 따름정리

- 서로 다른 소수 $p_j$에 대해 $m = p_1 \times p_2 \times \cdots \times p_r \times N^2$이라 하면 다음이 성립
    
    $n$을 두 제곱수의 합으로 나타낼 수 없음 $\Leftrightarrow n$은 어떤 $j = 1, \cdots, r$에 대해 소인수 $p_j \equiv 3 \pmod 4$를 가짐
