# 1. 수렴하는 수열

### 유계

- 모든 $n \in \mathbb{N}$에 대해 $|x_n| \leq M$을 만족하는 실수 $M > 0$이 존재할 때의 수열 $(x_n)$
- 수렴하는 수열은 모두 유계

### 극한과 사칙연산

$\lim a_n = a, \lim b_n = b$라 할 때 다음이 성립

- 모든 $c \in \mathbb{R}$에 대해 $\lim (ca_n) = ca$
- $\lim (a_n + b_n) = a + b$
- $\lim (a_nb_n) = ab$
- $\lim (\frac{a_n}{b_n}) = \frac{a}{b}$

# 2. 극한과 부등식

### 극한과 부등식

$\lim a_n = a, \lim b_n = b$라 가정

- 모든 $n \in \mathbb{N}$에 대해 $a_n \geq 0$이면 $a \geq 0$
- 모든 $n \in \mathbb{N}$에 대해 $a_n \leq b_n$이면 $a \leq b$
- 모든 $n \in \mathbb{N}$에 대해 $c \leq b_n$인 $c \in \mathbb{R}$이 존재하면 $c \leq b$
- 모든 $n \in \mathbb{N}$에 대해 $a_n \leq c$이면 $a \leq c$
