# 1. 소수에 대한 미해결 문제

### 쌍둥이 소수 추측

- $p, p + 2$가 모두 소수일 때, $p, p + 2$는 쌍둥이 소수

### 라그랑주의 추측

- 5보다 큰 모든 홀수는 소수 $p, q$에 대해 $p + 2q$의 형태로 쓸 수 있음

### 골드바흐의 추측

- 2보다 큰 모든 짝수는 두 소수의 합으로 표현 가능

### 골드바흐의 홀수 추측

- 5보다 큰 모든 홀수는 세 소수의 합으로 표현 가능

### 란다우의 추측

- $n^2 + 1$ 형태인 소수가 무한히 많이 존재

# 2. 소수의 분포

### 합성수 명제

- 임의의 양의 정수 $n$에 대해, 합성수이면서 연속인 정수가 $n$개 존재

# 3. $4n \pm 1$ 형태의 소수

### 정수 명제

- 모든 정수는 다음 형태 중 하나
    
    $4n, 4n + 1, 4n + 2, 4n + 3$
    
- $4n + 3$ 형태인 소수는 무한히 많이 존재

### 정수 보조정리

- $4n + 1$ 형태인 두 개 이상의 정수들의 곱은 같은 형태인 $4n + 1$ 형태로 표현 가능

# 4. 등차수열에서의 소수

### 디리클레의 정리

- $a, b$가 서로소인 양의 정수라 하면, 등차수열 $a, a + b, a + 2b, a + 3b, \cdots$는 무한하게 많은 소수 포함

# 5. 소수 정리

### 정수론

- 소수 $p$가 두 수의 제곱의 합이라면, $p$는 $4n + 1$ 형태
- $4n + 3$ 형태의 소수는 두 수의 제곱의 합 불가능

### 페르마 수

- $n = 0, 1, 2, 3, 4$에 대해 소수를 만드는 공식
    
    $F_n = 2^{2^n} + 1$
