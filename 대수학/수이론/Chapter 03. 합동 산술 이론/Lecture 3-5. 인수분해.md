# 1. 두 제곱수의 차

### 두 제곱수의 차

- 홀수인 정수 $n$을 두 제곱수의 차로 나타냄
    
    $n = a^2 - b^2 = (a - b)(a + b)$
    

# 2. 페르마 분해

### 페르마 분해

- $a_1 = \lceil \sqrt{n} \rceil$이라 하고, $a_1$은 $\sqrt{n}$보다 크거나 같으면서 $\sqrt{n}$에 가장 가까운 정수
- $a_1^2 - n$을 계산하여 완전제곱수가 나오면 $b_1^2 = a_1^2 - n$이라 함. 다음과 같이 인수분해 가능
    
    $n = a_1^2 - b_1^2 = (a_1 - b_1)(a_1 + b_1)$
    
- $a_1^2 - n$이 완전제곱수가 아니라면, 정수 $k$에 대해 $a_k^2 - n$이 완전제곱수가 될 때까지 $a_2 = a_1 + 1, a_3 = a_2 + 1, \cdots$ 시도
    
    $n = a_k^2 - b_k^2 = (a_k - b_k)(a_k + b_k)$
    

# 3. 합동 산술을 사용한 인수분해

### 자명하지 않은 인수

다음을 만족하면 정수 $d$가 정수 $n$의 자명하지 않은 인수

- $d \mid n$
- $d \neq 1, d \neq n$

### 인수분해 정리

- $a$와 $b$가 다음을 만족하는 정수라 하자
    
    $a^2 \equiv b^2 \pmod n, a \not\equiv \pm b \pmod n$
    
- $\gcd(a - b, n)$은 $n$의 자명하지 않은 인수
