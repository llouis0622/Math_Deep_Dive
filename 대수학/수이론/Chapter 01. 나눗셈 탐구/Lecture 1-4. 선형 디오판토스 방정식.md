# 1. 선형 디오판토스 방정식

### 일차 디오판토스 방정식

- 변수가 2개인 일반적인 $ax + by = c$의 형태, 변수 $x, y$는 정수이고 $a, b$는 0이 아님

# 2. 일반적인 디오판토스 방정식

### 디오판토스 방정식 명제

- $ax + by = c$에 대해 임의의 정수 $t$에 대해 $x = x_0 + bt, y = y_0 -at$도 방정식의 해

$\gcd(a, b) = g$라 할 때, 다음이 성립

- $ax + by = c \Leftrightarrow g|c$

$\gcd(a, b) = g$일 때 $g|c$이고, $x_0, y_0$가 방정식 $ax + by = c$의 특수해이면, 방정식의 모든 다른 해는 다음과 같음

- 정수 $t$에 대해, $x = x_0 + (\frac{b}{g})t, y = y_0 + (\frac{a}{g})t$

### 디오판토스 방정식 따름정리

$\gcd(a, b) = 1$이고, $x_0, y_0$가 방정식 $ax + by = c$의 특수해이면, 방정식의 모든 다른 해는 다음과 같음

- 임의의 정수 $t$에 대해 $x = x_0 + bt, y = y_0 - at$
