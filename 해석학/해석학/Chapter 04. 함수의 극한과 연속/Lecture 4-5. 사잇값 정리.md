# 1. 사잇값 정리

### 사잇값 정리

- 연속함수 $f : [a, b] \rarr \mathbb{R}$와 $f(a) < L < f(b)$ 또는 $f(a) > L > f(b)$를 만족하는 실수 $L$에 대해 $f(c) = L$인 점 $c \in (a, b)$가 존재

# 2. 연결집합의 보존

### 연결집합의 보존

- 함수 $f : G \rarr \mathbb{R}$가 연속이라고 할 때 집합 $E \subseteq G$가 연결집합이면 $f(E)$도 연결집합

# 3. 완비성

### 완비성 공리를 이용한 증명

- 연속함수 $f$에 대해 $f(a) < 0 < f(b)$인 특수한 경우에서 어떤 $c \in (a, b)$에 대해 $f(c) = 0$임을 보임
- $K = \{x \in [a, b] \mid f(x) \leq 0\}$
- $K$는 $b$에 의해 위로 유계, $a \in K$이므로 $K$는 공집합이 아님 → 완비성 공리에 의해 $c = \sup K$
- $f(c) > 0, f(c) < 0, f(c) = 0$
- $c$가 $K$의 상한이라는 사실로부터 처음 두 경우 배제 → $f(c) = 0$

### 축소구간성질을 이용한 증명

- $L = 0$이고 $f(a) < 0 < f(b)$인 특수한 경우에서 $I_0 = [a, b]$라 함
- 중점 $z = \frac{(a + b)}{2}$
- $f(z) \geq 0$이면 $a_1 = a, b_1 = z$
- $f(z) < 0$이면 $a_1 = z, b_1 = b$
- 각각의 경우에서 $f$는 구간 $I_1 = [a_1, b_1]$의 왼쪽 끝점에서 음수이고 오른쪽 끝점에서 음수가 아님
- 위 과정을 귀납적으로 반복하여 축소구간성질 사용 가능

# 4. 사잇값 성질

### 사잇값 성질

- 구간 $[a, b]$에 속한 모든 $x < y$와 $f(x)$와 $f(y)$ 사이의 모든 $L$에 대해 $f(c) = L$인 점 $c \in (x, y)$를 항상 찾을 수 있을 때, 함수 $f$는 구간 $[a, b]$에서 사잇값 성질을 가짐
