# 1. 극한으로서의 리만 적분

### 태그된 분할

- 분할 $P = \{x_0, x_1, x_2, \cdots, x_n\}$에서 각 부분구간 $[x_{k-1}, x_k]$마다 점 $c_k$를 하나씩 선택하는 조건을 추가한 것

### 태그된 분할로 생성된 리만 합

- 함수 $f : [a, b] \rarr \mathbb{R}$와 태그된 분할에 대해 $R(f, P) = \sum_{k=1}^n f(c_k)(x_k - x_{k-1})$

### $\delta$-세분

- $\delta > 0$에 대해 분할 $P$의 모든 부분구간 $[x_{k-1}, x_k]$이 $x_k - x_{k-1} < \delta$를 만족하는 $P$

### 리만 적분가능성에 대한 극한 판정법

- 유계 함수 $f : [a, b] \rarr \mathbb{R}$가 리만 적분가능하고 $\int_a^b f = A$이기 위한 필요충분조건은 임의의 $\epsilon > 0$에 대해 다음 조건을 만족하는 $\delta > 0$가 존재하는 것
    
    임의의 $\delta$-세분인 태그된 분할 $(P, \{c_k\})$에 대해 $|R(f, P) - A| < \epsilon$
    

# 2. 게이지와 $\delta(x)$-세분

### 게이지

- 모든 $x \in [a, b]$에 대해 $\delta(x) > 0$이면 함수 $\delta : [a, b] \rarr \mathbb{R}$를 $[a, b]$에서의 게이지라 함

### $\delta(x)$-세분

- 주어진 게이지 $\delta(x)$와 모든 부분구간 $[x_{k - 1}, x_k]$에 대해 $x_k - x_{k - 1} < \delta(c_k)$이면 태그된 분할 $(P, \{c_k\}_{n=1}^n)$을 $\delta(x)$-세분이라 함
    
    각 부분구간 $[x_{k-1}, x_k]$의 너비는 $\delta(c_k)$보다 작음
    

### 게이지 명제

- 구간 $[a, b]$에서 게이지 $\delta(x)$가 주어질 때, $\delta(x)$-세분인 태그된 분할 $(P, \{c_k\}_{n=1}^n)$이 존재

# 3. 일반화된 리만 적분가능성

### 일반화된 리만 적분값

- 다음이 성립할 때 함수 $f$가 구간 $[a, b]$에서 일반화된 리만 적분값 $A$를 가진다고 할 때, 임의의 $\epsilon > 0$에 대해 어떤 게이지 $\delta(x)$가 $[a, b]$에서 존재하여, $\delta(x)$-세분인 모든 태그된 분할 $(P, \{c_k\}_{n=1}^n)$에 대해 다음이 성립
    
    $|R(f, P) - A| < \epsilon$
    

### 일반화된 리만 적분값 명제

- 함수가 일반화된 리만 적분값을 가질 때, 그 적분값은 유일

### 디리클레 함수와 일반화된 리만 적분값

- 디리클레 함수 $g(x)$는 $[0, 1]$에서 일반화된 리만 적분이 가능하며 $\int_0^1 g = 0$

# 4. 미분적분학의 기본정리

### 미분적분학과 일반화된 리만 적분값

- $F : [a, b] \rarr \mathbb{R}$가 $[a, b]$의 모든 점에서 미분가능하다고 가정하고 $f(x) = F'(x)$로 둘 때, $f$는 다음과 같은 일반화된 리만 적분값을 가짐
    
    $\int_a^b f = F(b) - F(a)$
    

### 치환적분법

- 함수 $g : [a, b] \rarr \mathbb{R}$가 $[a, b]$의 각 점에서 미분가능하고 $F$가 집합 $g([a, b])$에서 미분가능하다고 가정할 때, 모든 $x \in g([a, b])$에 대해 $f(x) = F'(x)$이면 다음이 성립
    
    $\int_a^b (f \cdot g) \cdot g' = \int_{g(a)}^{g(b)} f$
