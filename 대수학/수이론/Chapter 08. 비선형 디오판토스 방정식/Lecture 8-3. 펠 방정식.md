# 1. 브라마굽타

### 브라마굽타

- 1년 이내에 방정식 $x^2 - 92y^2 = 1$을 풀 수 있다면 그는 수학자임

### 펠 방정식

- $x^2 - Ny^2 = 1$

# 2. 펠 방정식 풀기

### 펠 방정식 조건

- $x^2 - Ny^2 = 1$에서 정수 $N$은 $N > 1$이고 완전제곱수가 아님

### 펠 방정식 명제

- $N > 1$이고 완전제곱수가 아니라면 다음이 성립
    
    $x + y\sqrt{N} = a + b\sqrt{N} \Leftrightarrow x = a, y = b$
    
- 정수 $a, b, c, d, N$에 대해 다음 항등식이 성립함
    
    $(a^2 - Nb^2)(c^2 - Nd^2) = (ac + Nbd)^2 - N(ad + bc)^2$
    
- 무리수 $r = a + b\sqrt{N}$이 펠 방정식 $x^2 - Ny^2 = 1$의 해를 만든다면 역수 $\frac{1}{r} = r^{-1}$도 $x^2 - Ny^2 = 1$의 해를 만듦
- 무리수 $r = a + b\sqrt{N}$과 $s = c + d\sqrt{N}$이 펠 방정식 $x^2 - Ny^2 = 1$의 해를 만든다면 곱 $r \times s$도 $x^2 - Ny^2 = 1$의 해를 만듦
- $r = a + b\sqrt{N}$이 펠 방정식 $x^2 - Ny^2 = 1$의 해를 만든다면 $n = 1, 2, 3, \cdots$에 대해 $r^n = (a + b\sqrt{N})^n = \alpha + \beta\sqrt{N}$ 역시 해를 만듦

# 3. 펠 방정식의 기본 해

### 펠 방정식 명제

- $r = a + b\sqrt{N}, s = c + d\sqrt{N}$이 모두 펠 방정식 $x^2 - Ny^2 = 1$의 해를 만든다고 하면 다음이 성립
    
    $r < s \Leftrightarrow a < c$
    

### 펠 방정식 따름정리

- $r = a + b\sqrt{N}$이 펠 방정식 $x^2 - Ny^2 = 1$의 최소 양의 정수해를 만든다면 자연수 $n$에 대해 $r^n = (a + b\sqrt{N})^n$이 이 방정식의 모든 양의 정수해를 만듦
